# Coach Images Directory

Place the coach images in this directory:

## Required Images

### 1. **rajendran.jpg**
- Profile image of Dr. Rajendran Mani
- Dimensions: Recommended 400x600px (3:4 aspect ratio) or larger
- Format: JPG/PNG
- Path: `/coaches/rajendran.jpg`

### 2. **benjamin.jpg**
- Profile image of Benjamin Jerold
- Dimensions: Recommended 400x600px (3:4 aspect ratio) or larger
- Format: JPG/PNG
- Path: `/coaches/benjamin.jpg`

### 3. **together.jpg**
- Combined/highlight image of both coaches
- Dimensions: Recommended 600x600px (1:1 square) or landscape
- Format: JPG/PNG
- Path: `/coaches/together.jpg`

## Image Optimization Tips

- **Quality**: Use high-quality images (150+ DPI for print-ready, 72 DPI minimum for web)
- **Compression**: Optimize JPG/PNG files using tools like TinyPNG or ImageOptim
- **Format**: JPG for photos (best for compression), PNG for images with transparency
- **Size**: Keep file sizes under 500KB per image for optimal loading

## How Images Are Used

1. **Coach Profile Cards**: `rajendran.jpg` and `benjamin.jpg` are displayed as 3:4 aspect ratio cards with gradient overlays
2. **Highlight Banner**: `together.jpg` is displayed in a side-by-side layout with text content
3. **Responsive**: Images scale automatically for mobile and desktop

## Fallback Behavior

If images are missing:
- Coach cards will show a subtle gradient background
- Layout will remain intact and responsive
- Ensure images are properly named and placed for best visual presentation
